## SQL select

#### command

	SELECT column1, column2, ...
	FROM table_name;

column1, column2 자리에 데이터를 선택하고 싶은 필드 이름을 입력한다.
![select](select.png)
column은 `,`로 구별하며, 순서는 기재한 대로 나옴.

	SELECT * FROM table_name;

모든 필드를 다 선택하고 싶으면 `*`을 이용한다.
![select](select_all.png)
